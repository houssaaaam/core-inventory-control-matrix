## 🚀 The Product Vision (Why it Hooks Clients)

Traditional dashboard utilities force users to search through lines of rows to pinpoint supply errors or financial shifts. The **Core Inventory Control Matrix** fixes this user drop-off with three modern engineering principles:

*   **Predictive Insight Loop:** Features an automated, real-time ML/AI-style alert ticker that analyzes item velocities and active datasets instantly to display proactive restock directions.
*   **Glanceable Data Storytelling:** Employs an interactive Chart.js doughnut system with a responsive data center that maps your portfolio value breakdown instantly by categories.
*   **Sleek Micro-Interactions:** Uses a custom, low-latency layout built with glassmorphic layers, linear title text gradients, a breathing system pulse node, and self-dismissing notification toasts.

---

## ✨ Core Features

*   **Supabase Cloud Matrix Integration:** Real-time data pipeline syncing product variations seamlessly across all active clients.
*   **Granular RBAC System:** Built-in Role-Based Access Control allowing simulated tier shifts between **Admin**, **Manager**, and **Viewer** modes dynamically.
*   **Automated CSV Portfolio Export:** Direct spreadsheet serialization engine for rapid reporting and supply reviews.
*   **Contextual Selection Engine:** Advanced multi-select row framework allowing secure global batch purges.

The interface utilizes a sleek dark color spectrum, custom sub-pixel typography anti-aliasing (`Plus Jakarta Sans`), and lightweight vector favicon maps:
